Tentang Kami

Inovasi melalui Kode: Belajar Membangun Solusi Digital

Apa itu Barang kita?

Barang Kita adalah proyek akhir mata kuliah Pemrograman Berbasis Web yang dikembangkan oleh mahasiswa Universitas Syiah Kuala. Aplikasi ini dibuat sebagai wujud penerapan ilmu web development yang telah kami pelajari selama setengah semester.

Mengapa Kami Membuat Aplikasi Ini?

Tujuan Pembelajaran:
• Menerapkan konsep frontend development
• Belajar bekerja dalam tim
• Memahami alur pengembangan aplikasi web modern
• Mempraktikkan version control dengan Git
